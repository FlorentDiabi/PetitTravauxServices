Projet Nolark

Contexte

Projet scolaire réalisé dans le cadre du BTS SIO SLAM
L'entreprise Nolark est une boutique physique de vente de casques pour les deux roues motorisés. Elle dispose d'un site web et souhaite une prise en charge de la maintenance.

Notions découvertes :

  • Langages Web (HTML/CSS/JavaScript).
  • Responsabilités juridiques sur le Web.
  • Déploiment de service.

Missions

Responsabilités juridiques de l'entreprise :

La première mission a donc été de vérifier la conformité du site d'un point de vue respect de la réglementation en matière de collecte, traitement et stockage des données à caractère personnel (DCP) sous la forme d'une synthèse.

Ensuite, il a fallut utiliser le validateur du World Wide Web Consortium (W3C) afin de révéler les erreurs que contiennent le site afin de les corriger dans le futur.
